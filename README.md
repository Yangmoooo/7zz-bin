# 7zz-binaries

This repository automatically builds and packages `7zz`, the standalone version of 7-Zip console that supports all formats. The build is triggered whenever a new version of [7-Zip](https://github.com/ip7z/7zip) is released. A statically linked Linux version, `7zzs`, is also provided.
